## 2. Документація по розробленому веб-застосунку

### Призначення
Застосунок являє собою REST API сервіс для створення та зберігання текстових нотаток. Об'єкт нотатки містить поля `id`, `title`, `content` та `created_at`. Застосунок підтримує повернення даних як у форматі JSON, так і у вигляді простих HTML-таблиць залежно від заголовка `Accept`.
